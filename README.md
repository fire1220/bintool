# bintool
> 常用工具命令，使用Linux和macOS
## 命令说明

- dev 适用于git把当前分支的修改合并到dev(文件名，可以更改为其他名称)里,并且发布到jenkins里
- rancherlogin 用户rancher登录k8s服务器

## 案例

- dev命令：

  > 功能：
  >
  > ​	1.合并到文件名（dev）分支
  >
  > ​    2.如果添加-j或--jenkins参数，会通过jenkins发布，部署服务
